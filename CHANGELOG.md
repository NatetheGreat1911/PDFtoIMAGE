# Changelog

All notable changes to PDF to Image are documented here.

---

## [1.1] - 2026-04-19

### Added
- **Download tab** — scan any web page for PDF links and download them in bulk
- Drag-and-drop support for adding PDF files to the Convert queue
- Progress bar showing conversion status per file
- Option to choose output image format (PNG, JPEG)
- Option to set output DPI (72, 150, 300)

### Fixed
- App window now remembers its last size and position between sessions
- Improved error messaging when a PDF is password-protected

---

## [1.0] - 2026-04-17

### Added
- Initial release
- Batch convert PDF files to images (PNG)
- Simple drag-and-drop interface
- Windows standalone executable — no installation required
