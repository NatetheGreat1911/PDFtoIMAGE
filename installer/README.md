# Installer

This folder contains the [Inno Setup](https://jrsoftware.org/isinfo.php) script used to build the Windows installer.

**File:** `PDFtoImage_setup.iss`

To build the installer:
1. Install Inno Setup 6+
2. Open `PDFtoImage_setup.iss` in the Inno Setup IDE
3. Press **Build > Compile** (or F9)
4. The output installer will appear in the `Output\` folder
